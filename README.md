# red-tetris
Projet red-tetris from 42, Full Stack JS: nodejs, react, redux, communication with socket.io

## Usage:

### For developpements:
npm run dev-client\
npm run dev-server

Connexion on the port 8080

### To make a build
npm run build

### Launch the app in production mode (a build is required)
npm start

### Tests:
npm run eslint\
npm run test\
npm run coverage
